- MOSH session gives me "I have no name" error: for dnsinfo: see
    - The [mosh-server on OS X sometimes lacks a Mach context](https://github.com/mobile-shell/mosh/issues/249)
    - [Mac OS X mach bootstrap context (or something) is wrong](https://github.com/mobile-shell/mosh/issues/223)
    - Solution: Use `reattach-to-user-namespace` from [tmux-MacOSX-pasteboard](https://github.com/ChrisJohnsen/tmux-MacOSX-pasteboard) and this BASH script to wrap mosh-server
        ```
        #!/bin/bash
        set -euo pipefail
        export LANG=en_US.UTF-8
        ~/bin/reattach-to-user-namespace /opt/local/bin/mosh-server $@
        ```
